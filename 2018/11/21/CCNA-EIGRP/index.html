#CCNA# EIGRP协议

2018.11.21 D-Veda 计算机  热度 ℃

概要

EIGRP协议是Cisco开发且专用的高级距离矢量路由选择协议, 集链路状态路由选择协议和距离矢量协议路由选择的优点为一身
EIGRP协议适用于多种不同的拓扑和介质中, 是Cisco设备常用的路由选择协议之一
EIGRP协议能快速汇聚, 占用的带宽少, 支持多种网络层协议, 支持无线路由选择, 开销低, 负载均衡, 易于汇总

EIGRP协议

EIGRP(Enhanced Interior Gateway Routing Protocol)
EIGRP协议是标准的距离矢量协议
EIGRP协议使用带宽/延迟作为路径选择的标准
EIGRP协议的管理距离AD 90(Internal) 170(External)
EIGRP协议使用IP协议号88交换数据包, 使用内置RTP协议确保数据交换的可靠性
EIGRP协议是Cisco私有的协议(2013年已经公有化)
EIGRP协议使用组播224.0.0.10交换协议数据包
EIGRP协议不需要周期向邻居通告协议报文(RTP协议确保EIGRP协议报文安全)

EIGRP重要组件

  • EIGRP邻居发现机制
  • EIGRP弥散更新算法DUAL
  • EIGRP可靠传输协议RTP
  • EIGRP多协议模板

EIGRP邻居发现机制

由于RIP协议收敛速度慢, 被动等待计时器来收敛网络, 不能主动的识别网络故障
而出现了EIGRP邻居发现机制解决方案:

  • 主动的探测网络的可用性
  • 加快网络收敛
  • 增强网络稳定性

EIGRP邻居发现机制通过发送并交换Hello报文来建立, 维护, 终止邻居关系
EIGRP协议每隔5s发送Hello报文, 若15s没有收到邻居的Hello报文就取消建立邻居关系
EIGRP协议通过匹配参数(K值, AS号, 认证)来建立邻居关系
EIGRP协议的邻居发现机制会加快收敛, 主动识别网络, 增强EIGRP的会话稳定(完全删除一条路由最多需要15s)

EIGRP弥散邻居算法发(简介)

FD: 本地到达目标网段所花费的开销
AD: 邻居到达目标网段所花费的开销
FC(可行性条件): FS(AD) < S(FD)
FS: 满足FC条件的路由, 称为FS
S: 拥有最小FD的路由, 称为S

EIGRP协议配置命令

  • 注释

    router eigrp AS(eigrp区域的所有设备AS一定要相同, AS通常输入90)
    no auto-summary(关闭自动汇总)
    network 1.1.1.1 0.0.0.0(宣告本地直连的所有接口网段和自身虚拟接口的网段, 这里用Mask反掩码)

EIGRP协议相关的Show命令

show ip interface brief(查看接口状态及IP信息)
show running-config(查看所有配置)
show interfaces fastEthernet 0/0(查看接口fastEthernet 0/0的具体配置)
show running-config | section eigrp(查看EIGRP具体配置)
show ip eigrp neighbors(查看EIGRP协议邻居关系)
show ip eigrp topology(查看EIGRP协议数据库)
show ip protocols(查看协议信息)
show ip route(查看路由表信息)
show ip route eigrp(查看只包含EIGRP路由的路由表信息)

实例

  • 敲完协议后可以用show ip route和show running-config命令检查是否出现错误

小结
